# VellumVault Downloads

Public download page and release channel for VellumVault by Red Vellum Studio.

This repository intentionally contains only public distribution material:

- static download page
- privacy statement
- release notes through GitHub Releases
- Windows installer assets

It does not contain VellumVault source code, private campaign data, exported backups, restore points, attachments, or table notes.
