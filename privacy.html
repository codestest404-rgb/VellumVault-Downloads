Privacy Statement

VellumVault is designed as a local-first desktop application. The app does not require an account, subscription, cloud login, or telemetry service.

Information Collected By The App

VellumVault does not collect personal information for Red Vellum Studio. Campaign records, character sheets, notes, links, attachments, session logs, backups, and imported files are stored locally on the user's device unless the user chooses to export, copy, upload, or share them.

Local Storage

The Windows desktop app stores campaign data in a local SQLite database in the application data folder. Browser preview builds may use browser local storage for testing. JSON backups are created only when the user chooses to export them.

No Hosted Generation Services

VellumVault does not send campaign notes to hosted generation services. Recaps, notes, links, character sheets, backups, imports, and exports are handled locally by the desktop app.

D&D Beyond PDF Import

Character PDFs are parsed locally from files selected by the user. VellumVault does not log into, scrape, or connect to D&D Beyond accounts. Imported PDFs must be legally exported by the user from services they own or have permission to use.

Website Hosting

This public website is hosted through GitHub Pages. GitHub may receive standard web request information such as IP address, browser type, and request logs under GitHub's own policies. Red Vellum Studio does not add analytics, advertising trackers, or third-party marketing pixels to this site.

Contact

For privacy or safety questions about VellumVault, contact Red Vellum Studio through the GitHub repository owner.
