## Notes

Chronicle Ledger is a local-first desktop campaign manager for long-form homebrew and 5e-style tabletop campaigns. It includes campaign dashboards, NPCs, factions, locations, sessions, quest and hook tracking, party sheets, local D&D Beyond PDF import, DM Screen, Session Prep, Encounter Tracker, backups, restore points, and local SQLite storage.

Privacy posture:

- No account required.
- No cloud sync or telemetry.
- No hosted generation or model-provider integration.
- Installer builds contain the app only, not private campaign databases, backups, restore points, or notes.

Windows SmartScreen may warn on early unsigned builds because the installer is new. Verify downloads come from this repository and match the hash above.
